# apos-twitter

apostrophe-twitter is a widget for the [Apostrophe](http://github.com/punkave/apostrophe) content management system. apostrophe-twitter lets you add a Twitter feed to any content area created with Apostrophe. apostrophe-twitter is also intended as a demonstration of how to add custom widgets to Apostrophe and enable their use in your project. See [apostrophe-wiki](http://github.com/boutell/apostrophe-wiki) for a project that demonstrates how to use apostrophe-twitter.
